Introduccion al OVI diseños de sitios web

Bienvenido al Obejeto Virtual de Informatica OVI, qui econtratara contenidos de lectura y contenidos multimedia que le permitiran fortalecer el conocimiento y comprender el uso de html5 y css3.

El diseño web es el proceso de planificacion e implemenctacion de sitios web

Los conceptos que vamos a trabajr corresponde a las siguientes tematicas:

    Estructura básica página HTML (HTML, HEAD, BODY)
    Contenido en HTML (SECTION, ANSIDE, FOOTER, ARTICLE)
    Estilos internos y externos de CSS3
