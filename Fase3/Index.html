Introduccion al OVI diseños de sitios web con HTML y CSS3


Bienvenido al Objeto Virtual de Informatica OVI, aqui econtratara contenidos de lectura y contenidos multimedia que le permitiran fortalecer el conocimiento y comprender el uso de HTML5 y CSS3 para el desarrollo de paginas web.


Esta herramienta digital le ayudara a comprender el lenguaje HTML y en combinacion con las etiquetas de CCS3 permitira el despliegue de paginas interactivas.


HTML 5

Etiquetas generales y básicas HTML como Head, Body, Section, Footer

El lenguaje HTML permite lo siguiente:

  • Puede ser creado y editado con cualquier editor básico de textos.
  • Es multiplataforma, o sea, puede ser visualizado por cualquier navegador de cualquier sistema operativo.
  • Su lenguaje no diferencia entre mayúsculas y minúsculas.
  • Utiliza etiquetas o marcas, que consisten en breves instrucciones de comienzo y final, mediante las cuales se determina la forma en la que debe aparecer en su navegador el texto así como también las imágenes y demás elementos en la pantalla del ordenador.
  • Cada elemento de un documento HTML consta de una etiqueta de comienzo, un bloque de texto y una etiqueta de fin.
  • Lenguaje estático.
  • Es utilizado para la creación de páginas web.
  • Los documentos HTML son documentos de hipertexto que aparecen enlazando a otros documentos.
  • Es un estándar reconocido por todo el mundo y cuyas normas define un organismo sin animo de lucro llamado World Wide Web Consortium, más conocido como W3C. http://historiademihtml.blogspot.com/p/caracteristicas.html

Los conceptos que vamos a trabajr corresponde a las siguientes tematicas:
  • Estructura básica página HTML (HTML, HEAD, BODY)
  • Contenido en HTML (SECTION, ANSIDE, FOOTER, ARTICLE)
  • Estilos internos y externos de CSS3

CCS3

El formato CCS3 para aplicar estilos dinámicos y organizados en nuestras paginas web, para establecer una marca personal y única. CSS es un lenguaje para definir el estilo o la apariencia de las páginas web, escritas con HTML o de los documentos XML. CSS se creó para separar el contenido de la forma, a la vez que permite a los diseñadores mantener un control mucho más preciso sobre la apariencia de las página http://tc2-proyectodegradoingdesistemas.blogspot.com/2012/05/que-es-y-para-que-sirve-css3.html


Ventajas de usar CCS3

Al igual que HTML, no necesitas de ningún software costoso para empezar a codificar en CSS. Lo que sí necesitas es un amplio conocimiento del idioma si estás pensando en realizar tu código CSS en un bloc de notas. Aunque es posible, realmente no lo recomendamos pues no te permite distinguir tus etiquetas de tu contenido y si estás empleando muchos estilos distintos, entonces es mucho más complicado hallar errores. https://blog.aulaformativa.com/definicion-usos-ventajas-lenguaje-css3/


Diseño adaptativo.

Una de las mayores características de CSS3 que permite maximizar la experiencia de usuario en los dispositivos móviles es el uso de Media Queries. Las Media Queries o consultas de medios en CSS te permiten añadir estilos o reglas específicas según el tamaño de pantalla, la dirección del dispositivo o la densidad de pixeles.
